# Problem 15 - Shortest Path in Binary Grid

## Description
Given an N x N grid of 0s and 1s, find the shortest path 
from top-left (0,0) to bottom-right (N-1,N-1).
You can move in 8 directions. 0 = free, 1 = blocked.
Print path length or -1 if no path exists.

## Input Format
First line: N
Next N lines: N space-separated integers (0 or 1)

## Examples
Input:
3
0 0 0
1 1 0
1 1 0
Output: 4

## Constraints
1 <= N <= 100
Difficulty: Hard
Time Limit: 2 seconds
Memory Limit: 256MB
