# Problem 11 - Armstrong Number

## Description
Given an integer N, print Yes if it is an Armstrong number,
otherwise print No.
An Armstrong number equals the sum of its digits each raised
to the power of the number of digits.
Example: 153 = 1^3 + 5^3 + 3^3 = 153

## Input Format
Single integer N

## Output Format
Yes or No

## Examples
Input: 153
Output: Yes

Input: 123
Output: No

Input: 370
Output: Yes

## Constraints
1 <= N <= 10^6
Time Limit: 2 seconds
Memory Limit: 256MB
