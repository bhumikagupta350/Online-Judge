# Problem 7 - Palindrome Check

## Description
Given an integer N, print Yes if it is a palindrome,
otherwise print No.

## Input Format
Single integer N

## Output Format
Yes or No

## Examples
Input: 121
Output: Yes

Input: 123
Output: No

## Constraints
1 <= N <= 10^9
Time Limit: 2 seconds
Memory Limit: 256MB
