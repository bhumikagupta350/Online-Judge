# Problem 13 - Find Middle of Linked List

## Description
Given N nodes of a linked list, find and print the value 
of the middle node. For even length lists, print the 
second middle node.

## Input Format
First line: N (number of nodes)
Second line: N space-separated values

## Examples
Input:
5
1 2 3 4 5
Output: 3

Input:
4
1 2 3 4
Output: 3

## Constraints
1 <= N <= 10^4
Difficulty: Medium
Time Limit: 2 seconds
Memory Limit: 256MB
