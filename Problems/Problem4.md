# Problem 4 - Check Prime

## Description
Given an integer N, print Prime if it is prime, 
otherwise print Not Prime.

## Input Format
Single integer N

## Output Format
Prime or Not Prime

## Examples
Input: 7
Output: Prime

Input: 4
Output: Not Prime

## Constraints
1 <= N <= 10^6
Time Limit: 2 seconds
Memory Limit: 256MB
