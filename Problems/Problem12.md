# Problem 12 - Two Sum

## Description
Given an array of N integers and a target T, find two numbers 
that add up to T. Print their indices (0-based), smaller index first.
Exactly one solution is guaranteed.

## Input Format
First line: N T (array size and target)
Second line: N space-separated integers

## Examples
Input:
4 9
2 7 11 15
Output: 0 1

Input:
4 6
3 2 4 6
Output: 1 2

## Constraints
2 <= N <= 10^4
-10^9 <= nums[i] <= 10^9
Difficulty: Medium
Time Limit: 2 seconds
Memory Limit: 256MB
