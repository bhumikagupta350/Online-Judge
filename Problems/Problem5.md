# Problem 5 - Fibonacci Nth Term

## Description
Given N, print the Nth Fibonacci number.
Sequence starts: 0 1 1 2 3 5 8 13 21...

## Input Format
Single integer N

## Output Format
Nth Fibonacci number

## Examples
Input: 1
Output: 0

Input: 5
Output: 3

Input: 7
Output: 8

## Constraints
1 <= N <= 50
Time Limit: 2 seconds
Memory Limit: 256MB
