# Problem 9 - Find Maximum in Array

## Description
Given an array of N integers, print the maximum element.

## Input Format
First line: N
Second line: N space-separated integers

## Output Format
Maximum element

## Examples
Input:
5
3 7 1 9 4
Output: 9

Input:
4
-1 -5 -2 -3
Output: -1

## Constraints
1 <= N <= 10^5
-10^6 <= elements <= 10^6
Time Limit: 2 seconds
Memory Limit: 256MB
