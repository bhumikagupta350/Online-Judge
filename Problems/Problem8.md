# Problem 8 - Count Vowels

## Description
Given a string, print the count of vowels in it.
Vowels are a, e, i, o, u (lowercase only).

## Input Format
Single string with no spaces

## Output Format
Count of vowels

## Examples
Input: hello
Output: 2

Input: aeiou
Output: 5

Input: rhythm
Output: 0

## Constraints
Length of string <= 1000
Time Limit: 2 seconds
Memory Limit: 256MB
